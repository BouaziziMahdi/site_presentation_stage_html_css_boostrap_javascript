2eme activite : Configuration le progiciel Odoo et Pycharm

Définitions :

Odoo est un progiciel open source de gestion intégré qui contient un grand nombre de modules qui permettent de simplifier la gestion d’entreprise en générale et plus précisément est destiné à intégrer l'ensemble des données opérationnelles et de gestion de l'entreprise dans une base de données unique, accessible par une interface web
PyCharm est un environnement de développement intégré utilisé pour programmer en Python. Il permet l'analyse de code et contient un débogueur graphique. Il permet également la gestion des tests unitaires, l'intégration de logiciel de gestion de versions, et supporte le développement web « En peux dire donc que Odoo est un serveur de Pycharm »

Avantage :

Le principale avantage du Odoo qu’il permet de Il est possible d'intégrer d'autres systèmes comme le lecteur de code barre, par exemple au système ERP via une API (Application Programming Interface)...
